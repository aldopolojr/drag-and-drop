# Sortable Drag & Drop With Vanilla Javascript

## Notes
- draggable="true"
- :not()
- 'dragstart' event
- 'dragend' event
- 'dragover' event 
- e.clientY
- Node.insertBefore()
- .reduce()
- Element.getBoundingClientRect()
- Number.NEGATIVE_INFINITY or Number.POSITIVE_INFINITY
